## 📱 移动端专属功能

### 底部导航
```javascript
// 自动检测移动设备
if (window.innerWidth < 768) {
    // 显示底部导航
}
```

### 触控优化
```css
* {
    -webkit-tap-highlight-color: transparent; /* 移除点击高亮 */
    -webkit-user-select: none; /* 禁用选中文本 */
}
```

### 响应式设计
```css
@media (min-width: 768px) {
    /* 平板和电脑隐藏移动端元素 */
    .bottom-nav, .fab {
        display: none;
    }
}
```
