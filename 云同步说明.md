# 任务本云同步使用指南

## 问题说明

由于浏览器 CORS 安全限制，从 GitHub Pages 访问 Gitee API 会被阻止。

## 解决方案

### 方法 1：本地使用（推荐）

**电脑：**
- 直接双击 `D:\launch-app.bat` 打开
- 登录 Gitee 云同步
- 数据自动保存到 Gitee

**手机：**
- 需要搭建本地服务器（参考 `qrcode.html`）
- 或者使用下面方法 2

### 方法 2：导出/导入（备用方案）

如果云同步暂时不能用，可以用：

1. **导出数据**：点击「云同步」区域的「导出」按钮
2. **传输**：通过微信/网盘发送导出的文件
3. **导入**：在另一台设备点击「导入」按钮

### 方法 3：等待 Gitee Pages

正在联系 Gitee 开通 Pages 服务，开通后：
- 手机访问 `https://kuangquanshui73.gitee.io/tasks-web`
- 可以直接用 Gitee 云同步
