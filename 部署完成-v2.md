## 🔧 故障排查

### 手机上看不到新版本？

**解决方法：**
1. **清除缓存**
   - iPhone: 设置 → Safari → 清除历史记录与网站数据
   - Android: Chrome → 设置 → 隐私 → 清除浏览数据

2. **使用无痕模式**
   - iPhone: Safari → 无痕浏览
   - Android: Chrome → 新建无痕标签页

3. **等待更长时间**
   - GitHub Pages 有时需要 10-15 分钟

### 底部导航栏没有反应？

**解决方法：**
1. 确认访问的是 https://kuangquanshui73-create.github.io/tasks/
2. 强制刷新页面（Ctrl + F5）
3. 清除浏览器缓存

### 云同步不工作？

**解决方法：**
1. 点击"⚙️ 设置"标签
2. 检查云同步状态
3. 如果显示"未连接"，点击登录
4. 确保 Token 正确
