# 🌟 云端同步使用指南 🌟

太棒了!你的任务本现在支持**自动云端同步**了!💖

## ✨ 什么是云端同步?

之前你的数据只保存在浏览器里,换设备就看不到。现在:

**之前:** 📱手机 ➜ ❌ 电脑看不到数据  
**现在:** 📱手机 ➜ ☁️云端 ➜ ✅ 电脑也能看到!

---

## 🚀 如何使用云端同步?

### 第一步:登录GitHub账号

1. 打开任务本,向下滚动找到 **"☁️ 云端同步 ☁️"** 区域
2. 点击 **"🐙 GitHub登录"** 按钮
3. 会弹出GitHub授权页面
4. 点击"Authorize"授权(这只是授权Gist功能,不会影响你的GitHub)
5. 登录成功!

> **注意:** 第一次登录时,你需要有一个GitHub账号(免费注册)

---

### 第二步:享受自动同步! 🎉

登录后,系统会自动:

✅ **每5分钟自动同步** - 不用管它,它会自己同步  
✅ **页面关闭前同步** - 确保数据不丢失  
✅ **合并数据** - 多设备使用时,自动合并所有任务  
✅ **实时显示** - 显示最后同步时间

---

## 🔧 技术原理(想了解的话)

**GitHub Gist是什么?**
- GitHub提供的代码片段存储服务
- 类似云文档,可以存储文本数据
- 完全免费,非常稳定

**如何工作的?**
```
任务本 → GitHub API → 你的私人Gist → 云端
  ↓                                    ↑
  ↓←←←←←←← 自动同步 ←←←←←←←←←←←←←←←←
```

**数据存储位置:**
- 任务数据: `tasks.json`
- 用户数据: 星星、打卡天数等
- 都在你的GitHub账号下的Gist里
