# 🌟 小可爱任务本 🌟

一个超级可爱、超级好用的任务管理系统!让完成任务变得像玩游戏一样有趣!

## ✨ 特色功能

### 💖 超级可爱的界面
- 粉色系温馨配色,治愈又可爱
- 滚动的渐变背景动画
- 弹跳的成就徽章
- 完成任务时的撒花庆祝动画
- 温馨的问候语(根据时间自动变化)

### 🎯 简单易用的操作
- **快速添加**: 输入任务名称,一键添加
- **自动时间**: 自动填充当前时间和1小时后的截止时间
- **一键完成**: 点击圆圈即可完成任务
- **筛选标签**: 快速查看全部/进行中/已完成任务

### 📝 智能任务管理
- **三级优先级**: 🟢悠闲、🟡普通、🔴重要
- **备注功能**: 为每个任务添加详细备注
- **截止时间**: 精确到日期和时间
- **提醒功能**: 任务到期前30分钟自动提醒

### 🎉 有趣的完成体验
- **复盘感想**: 完成任务后写下你的感受和收获
- **奖励系统**: 选择预设奖励或自定义奖励
- **星星奖励**: 根据任务优先级获得星星(高优先级5星,中3星,低1星)
- **撒花动画**: 完成任务时触发可爱的撒花效果

### 🏆 成就系统
- **总完成数**: 记录你完成的所有任务
- **连续打卡**: 每天完成任务增加打卡天数
- **星星收集**: 累积星星兑换奖励

### 🎁 奖励商店
用你获得的星星兑换各种奖励:
- ☕ 喝杯奶茶 (5⭐)
- 🎬 看部电影 (10⭐)
- 🎮 玩一小时游戏 (15⭐)
- 🛍️ 买个小礼物 (20⭐)
- 🍽️ 吃顿大餐 (30⭐)
- ✈️ 计划一次小旅行 (50⭐)

### 📊 每日统计
- 今日任务数
- 今日已完成
- 今日剩余
- 今日获得的星星

## 🎮 如何使用

### 添加任务
1. 在"添加新任务"框中输入任务名称
2. 选择优先级(默认为普通)
3. 设置截止时间(自动填充当前时间+1小时)
4. 勾选"提醒我"(可选)
5. 点击 ➕ 按钮

### 完成任务
1. 点击任务左侧的圆圈 ⭕
2. 在弹出的窗口中:
   - 查看获得的星星数
   - 写下任务完成感想(可选)
   - 选择奖励建议或输入自定义奖励(可选)
3. 点击"确认完成"或"跳过"
4. 享受撒花庆祝!🎉

### 编辑任务
1. 点击任务右侧的 ✏️ 按钮
2. 修改任务信息、备注、优先级、截止时间
3. 点击"保存修改"

### 删除任务
1. 点击任务右侧的 🗑️ 按钮
2. 确认删除

### 兑换奖励
1. 滚动到"奖励商店"
2. 找到你想要的奖励
3. 如果星星足够,点击"兑换"
4. 享受你的奖励!

## 💾 数据保存

所有数据自动保存在浏览器本地:
- 任务列表
- 完成记录
- 星星数量
- 连续打卡天数
- 已兑换的奖励

刷新页面不会丢失任何数据!

## 🌈 特殊功能

### 时间问候
系统会根据不同时间段显示不同的问候语:
- 🌅 早上 (5:00-9:00): 早安呀,新的一天开始啦!
- ☀️ 上午 (9:00-12:00): 上午好,元气满满!
- 🍽️ 中午 (12:00-14:00): 中午好,记得好好吃饭哦!
- 💪 下午 (14:00-18:00): 下午好,继续加油!
- 🌙 晚上 (18:00-22:00): 晚上好,辛苦了一天啦!
- 😴 深夜 (22:00-5:00): 夜深了,早点休息哦!

### 逾期提醒
逾期任务会显示红色闪烁提示,提醒你尽快完成!

### 连续打卡
每天完成至少一个任务即可增加打卡天数,中断后重新计算!

## 🎨 设计理念

这款任务管理系统采用了"游戏化"设计理念:
- **可爱的视觉**: 用粉色和卡通元素营造轻松氛围
- **即时反馈**: 完成任务立即获得奖励和动画
- **成长系统**: 星星和成就激励持续使用
- **自我奖励**: 培养自我关爱和正面强化

让完成任务不再枯燥,而是一种享受!
