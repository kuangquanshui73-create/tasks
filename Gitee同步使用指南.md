## 🚀 快速开始(3步搞定!)

### 第一步:注册Gitee账号(如果没有)

1. 访问: https://gitee.com
2. 点击"注册"
3. 填写用户名、邮箱、密码
4. 完成!(几分钟就能搞定)

### 第二步:获取访问令牌

1. 访问: https://gitee.com/profile/personal_access_tokens
2. 点击"生成新令牌"
3. 填写:
   - **令牌描述**: 任务本同步(随便填)
   - **权限**: 勾选 `repositories` 和 `user_info`
4. 点击"提交"
5. **复制令牌** (格式像: `gitee_xxxxxxxxxxxxxxxxxxxxxxxxx`)
   > ⚠️ 重要:只显示一次,务必复制保存!

### 第三步:在任务本中登录

1. 打开任务本,向下滚动找到 **"☁️ 云端同步"** 区域
2. 点击 **"🐧 Gitee登录"** 按钮
3. 填写:
   - **访问令牌**: 粘贴刚才复制的令牌
   - **Gitee用户名**: 你的Gitee用户名
4. 点击"登录"
5. 成功!✅

就这么简单!

## 🔧 技术原理

**数据存储在哪里?**
- 你的Gitee私有仓库 `cute-tasks-backup`
- 文件名: `tasks.json`

**如何同步的?**
```
任务本 → Gitee API → 你的仓库 → 云端
  ↓                           ↑
  ←←← 自动同步 ←←←←←←←←←←←←
```

**什么是访问令牌?**
- 类似于密码,但更安全
- 可以随时撤销
- 只授予必要的权限
