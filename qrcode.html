手机访问指南

让手机也能随时打开小可爱任务本

🌐 在手机浏览器输入这个地址
局域网访问地址（示例）
http://192.168.x.x/index.html
把 192.168.x.x 替换成你电脑的真实 IP
💡 手机和电脑必须连接同一个 WiFi，才能通过局域网互访。
📂 直接双击打开 HTML 文件只能在本机用，不能跨设备。手机访问需要搭建一个简易 HTTP 服务。
🔍 第一步：查找本机 IP 地址
  • 1
    按 Win + R，打开运行窗口
    同时按住键盘上的 Windows 键和 R 键
  • 2
    输入 cmd，按回车
    打开命令提示符（黑色窗口）
  • 3
    在黑色窗口中输入以下命令
    ipconfig
    找到「以太网适配器」或「无线局域网适配器 WLAN」下的 IPv4 地址，通常以 192.168 开头
🚀 第二步：启动简易 HTTP 服务器
  • A
    方法一：用 Python（推荐）
    如果电脑装了 Python，在任务本文件夹空白处按住 Shift + 右键，选「在此处打开命令行」，输入：
    # Python 3
    python -m http.server 8080
    然后手机访问：http://你的IP:8080/index.html
  • B
    方法二：用 Node.js
    npx serve -p 8080 D:\工作安排管理系统
    然后手机访问：http://你的IP:8080
  • C
    方法三：用 VS Code Live Server 插件
    安装 Live Server 扩展 → 右键 index.html → 选「Open with Live Server」→ 手机访问提示的局域网地址
📲 第三步：添加到手机主屏幕
  • iOS
    Safari 浏览器打开页面
    点底部「分享」按钮 → 选「添加到主屏幕」→ 像 App 一样从桌面打开 🎉
  • 安卓
    Chrome 浏览器打开页面
    点右上角三个点菜单 → 选「添加到主屏幕」或「安装应用」→ 桌面出现图标
🏠 在本机打开任务本 ☀️ 打开每日早报
