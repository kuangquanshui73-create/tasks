## 🔧 故障排查

### 如果手机上看不到任务

**1. 检查 GitHub Pages 是否更新**
- 打开 https://kuangquanshui73-create.github.io/tasks/
- 按 F12 打开开发者工具
- 在 Console 输入：`cloudSyncManager.branch`
- 应该返回 `"master"`，如果返回 `"main"` 说明还是旧版本

**2. 清除浏览器缓存**
- Chrome: 设置 → 隐私 → 清除缓存 → 全部时间
- Safari: 设置 → Safari → 清除历史记录与网站数据
- 或直接使用无痕/隐私模式测试

**3. 等待更长时间**
- GitHub Pages 有时需要 10-15 分钟
- 可以用工具测试：https://gitee.com/kuangquanshui73/tasks

### 如果登录失败

**使用登录测试工具：**
1. 打开 `login-test.html`
2. 点击 "开始测试"
3. 查看测试结果

**常见原因：**
- Token 无效或已过期
- 仓库名称不对（应该是 `cute-tasks-backup`）
- Token 权限不足（需要 `repositories` 权限）
