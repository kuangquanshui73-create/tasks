# 🌟 小可爱任务本 · 使用习惯建议

> 工具只有用起来才有价值。这份指南帮你养成真正管用的使用习惯，让任务本成为你每天的好伙伴。

---

## ☀️ 一、早上打开任务本的习惯

### 建立「晨间仪式感」
把打开任务本变成早晨的第一个动作——比起打开社交媒体，这会让你的一天更有掌控感。

**推荐流程（5分钟搞定）：**
1. 坐下来、倒杯水（或咖啡）
2. 双击桌面的「启动任务本.bat」，打开 **每日早报页面**（daily-brief.html）
3. 看看今天日期和倒计时，感受一下「今天还有多少时间」
4. 读一条励志句子，给自己加油
5. 浏览今日待办，在脑子里过一遍「今天最重要的3件事」
6. 点击「开始今天的任务」，进入任务本正式开始

### 小技巧
- 把 `启动任务本.bat` 的快捷方式放到桌面或任务栏，做到**一秒启动**
- 手机用户：把任务本网址添加到主屏幕，跟手机 App 一样方便
- 不要在还没看任务本之前就开始处理邮件或消息——先规划，再执行

## 🌙 二、每天结束时的复盘方法

### 为什么要复盘？
不复盘，今天的经验就白费了。每晚5分钟的复盘，能让你越做越好。

**推荐「三问复盘法」：**

| 问题 | 要思考的内容 |
|------|------------|
| ✅ 今天完成了什么？ | 看任务本，数一数完成的数量，给自己肯定 |
| ❌ 什么没完成？为什么？ | 是任务太难、时间不够，还是分心了？ |
| 🔄 明天怎么调整？ | 把未完成的移到明天，调整优先级，拆分大任务 |

**操作步骤：**
1. 晚上8点前（截止时间）打开任务本
2. 把今天完成的任务勾掉，享受那一刻的成就感
3. 把没完成的任务检查一遍，决定：延期 / 删除 / 拆分
4. 为明天添加2-3个最重要的任务
5. 关闭前看一眼「连续打卡」天数，激励自己明天继续

### 小技巧
- 设一个**晚间提醒**（手机闹钟），提示自己「该复盘了」
- 复盘不是为了批判自己，而是为了更好地安排明天
- 如果今天完成率很低，先问原因，再调整任务数量——也许是计划太满了

## 🎯 三、如何用好专注模式

专注模式（番茄钟）是对抗拖延的核武器——把大任务分成25分钟小块，大脑会更愿意开始。

### 专注模式使用步骤
1. 在任务列表里选一个要做的任务，点击进入**专注模式**
2. 设定好专注时长（推荐25分钟）
3. **手机调静音，关闭无关网页**，只保留任务本
4. 专注计时开始，全力做这一件事
5. 时间到，休息5分钟（站起来走走、喝点水）
6. 下一轮开始前，可以继续当前任务或切换新任务

### 高效专注的关键原则

**✦ 任务要足够小**
不要对着「写一份报告」开始专注——太大了根本无从下手。  
把它拆成：「写报告提纲」「写第一节」「整理数据」……每块25分钟内能完成。

**✦ 开始前清空干扰**
- 把手机屏幕朝下放，或开启勿扰模式
- 告诉身边的人「我在专注XX分钟」
- 把想到的其他事情快速写进任务本，别让大脑牵挂它们

**✦ 完不成也没关系**
番茄钟到点了但任务还没做完？很正常。下一轮继续，或者拆得更小一些。

## 🔥 四、如何维持连续打卡

连续打卡天数是最强的「沉没成本激励」——舍不得断掉的那种。

### 打卡习惯的养成期（前21天最关键）

**前3天：超低门槛开始**
- 每天只要求自己「打开任务本 + 完成1件事」
- 再小的任务也算完成，哪怕是「喝一杯水」
- 目标是**建立仪式感**，不是立刻高效

**第4-14天：建立关联**
- 把任务本和某个固定的日常习惯绑定（比如喝早咖啡时打开）
- 每天完成≥2件任务才算打卡成功（逐步提高标准）
- 关注连续天数，让数字本身成为动力

**第15-21天：固化习惯**
- 这时候打开任务本应该已经是自动化动作了
- 开始关注「完成率」，尝试每天清空全部任务

### 不要断掉的秘诀

| 场景 | 建议 |
|------|------|
| 今天特别忙 | 至少完成1件任务，哪怕只是回复一封邮件 |
| 出差或旅行 | 手机打开任务本，完成一件小事 |
| 打卡断了 | 不要放弃！立刻重新开始，记录新的连续天数 |
| 没有动力 | 去看看「奖励商店」，想想攒点数能换什么 |

## 🎁 五、如何用好奖励商店让自己坚持

奖励机制是把枯燥任务变成有趣游戏的关键。

### 奖励设置原则

**✦ 奖励要真实、要诱人**
不要设置虚假奖励——只设置你真的想要的东西。

**好的奖励例子：**
- 🎮 玩1小时游戏（花费 20 积分）
- 🍰 买一块喜欢的蛋糕（花费 30 积分）
- 📺 看一集喜欢的剧（花费 15 积分）
- 🛒 买一件心水了很久的东西（花费 100 积分）
- 😴 睡个懒觉到10点（花费 25 积分）

**✦ 分层设置，大小奖励都有**
- 小奖励（10-30积分）：频繁可得，维持日常动力
- 大奖励（50-200积分）：长期目标，值得为之坚持

**✦ 兑换要立刻执行**
用积分兑换了奖励，就马上去享受——延迟兑现会让奖励失效。

### 积分获取策略
- **日常任务**：每完成一件任务 +积分
- **连续打卡**：7天 / 30天连续打卡有额外奖励
- **困难任务**：给高难度任务设置更高积分值，做完更爽

### 遇到「不想兑换」的情况
如果积分攒了很多但不想花，说明奖励设置得不够吸引人——回去更新奖励商店，换成现在更想要的东西。
